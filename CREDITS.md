## The Story

This project started with a simple idea: *"I watch long YouTube lectures for exam prep, but rewatching everything for revision is impossible. Can we convert them into PDF study material with screenshots and transcript context?"*

What began as a basic PDF export tool grew into a full-featured study companion through daily collaboration — brainstorming features, testing edge cases, fixing bugs, and adding exports for every major AI platform (NotebookLM, Gemini, Claude, ChatGPT, Ollama, Anki).

Every feature in this app was shaped by real study needs:

- **Visual Transcript PDF** → because multimodal AI works best when it reads the text first, then sees the image
- **13 export formats** → because different study workflows need different tools
- **Hindi/English support** → because Indian education is bilingual
- **8GB RAM target** → because not everyone has a high-end machine
- **Crash recovery** → because 4-hour lectures shouldn't be lost to a crash
